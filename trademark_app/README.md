# 상표 유사성 검토 시스템

특허법인 내부용 상표 선행조사 자동화 도구입니다.  
KIPRIS Open API 와 연동하여 G코드/S코드 분리 검색, 결과 합산, 보고서 자동 작성을 지원합니다.

---

## 빠른 시작

### 1단계 — Python 설치 확인

```
py --list
```
`-3.13` 이상이 목록에 있어야 합니다.  
없으면 https://www.python.org/downloads/ 에서 Python 3.13 설치.

### 2단계 — 패키지 설치

```
py -3.13 -m pip install -r requirements.txt
```

### 3단계 — 환경 설정

프로젝트 폴더의 `.env` 파일을 열어 KIPRIS API 키를 입력합니다.

```
KIPRIS_API_KEY=발급받은_키_입력
OFFICE_NAME=특허법인 OO
```

> API 키 발급: https://www.data.go.kr → "특허청 상표 출원정보" 검색 → 활용신청

### 4단계 — 실행

**방법 A: 더블클릭**  
`start.bat` 파일을 더블클릭합니다.

**방법 B: 터미널**
```
py -3.13 -m streamlit run app.py
```

브라우저에서 http://localhost:8501 접속

## 기능 안내

| 기능 | 설명 |
|------|------|
| 5단계 위저드 | 상표명 → 상품 선택 → 검색식 → 결과 → 보고서 |
| 복수 코드 검색 | G코드/S코드 분리 검색 후 중복 제거 합산 |
| 유사도 계산 | 바이그램 기반 자동 유사도 점수 (0~100%) |
| 위험도 판정 | 고위험(70%↑) / 중위험(50%↑) / 저위험 |
| 보고서 | 검토 의견 자동 초안 + TXT 다운로드 |
| 케이스 관리 | 검토 이력 세션 내 저장 및 재열람 |
| CSV 내보내기 | 검색 결과 CSV 다운로드 |

## 화면 구성

```
사이드바
├── 🏠 홈 (새 검토)
├── 📋 검색 히스토리
├── 📊 케이스 관리
├── 📄 보고서 관리
├── ⚙️  설정
└── ❓  사용 가이드

메인 — 5단계 위저드
STEP 1  상표명 입력 + 자동 분석 + 변형어 미리보기
STEP 2  지정상품 검색·선택 (자동완성, 45류 전체)
STEP 3  키프리스 검색식 확인 및 수정
STEP 4  G/S 코드 분리 검색 결과 + 유사도 테이블
STEP 5  검토 의견 작성 + 케이스 저장 + 보고서 다운로드
```

---

## 테스트 방법

1. `start.bat` 실행
2. STEP 1 → **POOKIE** 입력 → 다음
3. STEP 2 → **티셔츠** 검색 → 선택(25류) → 다음
4. STEP 3 → 검색식 확인 → 검색 실행
5. STEP 4 → **G코드 N건 + S코드 M건 = 총 T건** 확인

## API 키 미설정 시

`.env`에 키가 없으면 자동으로 **Mock 데이터**로 동작합니다.  
결과 상단에 `🟡 Mock 데이터` 배지가 표시됩니다.  
실제 API 연결 후 `🟢 실제 KIPRIS` 배지로 바뀝니다.
